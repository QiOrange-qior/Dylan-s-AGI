# **AGI·智造未来 \- 个人作品集网站**

欢迎来到 “AGI·智造未来” 个人作品集网站的项目说明。这是一个专为高级AI产品经理 Dylan Chang (气橙) 设计和开发的现代化单页网站。

本网站旨在以一个干净、优雅且富有交互性的界面，系统地展示作者在AI产品策略、项目管理、游戏创新及数据自动化等领域的专业技能与实践成果。设计灵感源于苹果（Apple）的设计语言，追求极致的细节和用户体验。

## **✨ 主要功能**

* **优雅的单页布局**：所有内容整合在同一页面，通过平滑滚动的导航栏，用户可以轻松访问“首页”、“技能”、“项目”和“联系方式”等各个部分。  
* **Apple风格设计**：采用现代化的UI设计，包括毛玻璃（Glassmorphism）效果的卡片、动态渐变背景和微妙的动画，创造出精致的视觉体验。  
* **明/暗主题切换**：内置一键切换浅色和深色模式的功能，并能根据用户的系统偏好自动选择初始主题。用户的选择会通过 localStorage 被浏览器记住。  
* **动态项目筛选**：在“精选项目”区域，用户可以根据“AIGC 产品”、“AI 游戏创新”、“数据与自动化”等分类对项目卡片进行实时筛选。  
* **全方位响应式设计**：网站布局能够完美适配从桌面显示器到移动设备的各种屏幕尺寸，确保在任何设备上都有一致的浏览体验。  
* **丰富的微交互**：包括卡片悬浮效果、3D旋转头像、能量环动画等，增加了网站的趣味性和互动性。

## **🛠️ 技术栈**

该项目完全基于前端原生技术构建，不依赖任何外部框架，以确保最佳的性能和可维护性。

* **HTML5**: 负责网站的语义结构。  
* **CSS3**:  
  * 使用 **CSS 变量 (Variables)** 构建了灵活的设计系统，便于全局样式的管理和主题切换。  
  * 采用 **Flexbox** 和 **Grid** 进行现代化布局。  
  * 通过 **Keyframes** 实现所有自定义动画效果。  
* **原生 JavaScript (ES6+)**:  
  * 实现项目筛选逻辑。  
  * 实现明/暗主题切换功能。  
  * 控制元素的动态显示和交互。

## **🚀 如何开始**

由于这是一个完全自包含的单文件项目，使用起来非常简单：

1. 克隆或下载此仓库。  
2. 直接在任何现代浏览器中打开 index.html 文件即可查看。

## **🎨 如何定制**

您可以轻松地将此模板修改为您自己的个人网站：

1. **修改个人信息**:  
   * **头像**: 在 hero-section 部分，找到 .avatar-core-hero 元素并替换 \<img\> 标签中的 src 链接。  
   * **姓名/标题**: 修改 h1 和 p.hero-handle 中的文本内容。  
   * **个人简介**: 修改 p.hero-subtitle 中的文本。  
   * **社交链接**: 在 footer 部分，替换 \<a\> 标签中的 href 链接。  
2. **更新技能与项目**:  
   * **技能**: 在 skills 部分，直接修改 .glass-card 内的文本内容和标签。  
   * **项目**: 在 projects 部分，您可以复制或修改 .project-card-container 元素块来更新您的项目。  
     * data-category 属性用于筛选，请确保其值为 aigc, game, 或 automation 之一。  
     * 替换项目图片、标题、描述和标签。  
3. **调整设计风格**:  
   * **主色调**: 在 \<style\> 标签顶部，修改 :root 中的 CSS 变量值，如 \--brand-pink, \--brand-purple, 和 \--brand-blue，即可轻松更改网站的主题颜色。
